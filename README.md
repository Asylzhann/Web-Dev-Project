# LibApp: Library Management Application 📚

## Project Overview 🌟

### Key Features:

- **User Registration** 📝: Allows users to create and manage their accounts.
- **Book Search** 🔍: Users can search for books in the library.
- **Book Borrowing** 📚: Users can borrow books and manage their borrow history.
- **Book Return** 🔄: Users can return borrowed books and update their accounts.
- **Account Management** 👤: Users can view and edit their account details.
